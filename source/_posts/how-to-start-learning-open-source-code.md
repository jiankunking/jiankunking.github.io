* 通过文档了解项目
	* 这个项目是干什么的?
	* 能解决哪些问题?
	* 适合在哪些场景使用?
	* 有哪些功能?
	* 如何使用?
* 用以点带面的方式来阅读源码
	* 最好是带着问题阅读源码，最好是带着问题答案去阅读源码
		* RocketMQ的消息是怎么写到文件里的?
		* Kafka的Coordinator是怎么维护消费位置的?
